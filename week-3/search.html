Searching & Sorting Algorithms

There are mainly two types of search:

  • Linear search
  • Binary search

Linear search is a simple search algorithm that checks each element of a list sequentially until it finds the target value.

Binary search is a search algorithm that works by comparing the target value to the middle element of the list. If the target value is equal to the middle element, then the position is found. Otherwise, if the target value is lower than the middle element, then the search continues on the lower half of the list. If the target value is greater than the middle element, then the search continues on the upper half of the list. It is used when the list is sorted.

The following code demonstrates the linear search algorithm:

    
      function linearSearch(arr, target) {
        for (var i = 0; i < arr.length; i++) {
          if (arr[i] === target) {
            return i;
          }
        }
        return -1;
      }

The following code demonstrates the binary search algorithm:

    
      function binarySearch(arr, target) {
        var low = 0;
        var high = arr.length - 1;
        var mid;
        while (low <= high) {
          mid = Math.floor((low + high) / 2);
          if (arr[mid] < target) {
            low = mid + 1;
          } else if (arr[mid] > target) {
            high = mid - 1;
          } else {
            return mid;
          }
        }
        return -1;
      }

The following code demonstrates the binary search algorithm with a recursive function:

    
      function binarySearch(arr, target, low, high) {
        if (low > high) {
          return -1;
        }
        var mid = Math.floor((low + high) / 2);
        if (arr[mid] < target) {
          return binarySearch(arr, target, mid + 1, high);
        } else if (arr[mid] > target) {
          return binarySearch(arr, target, low, mid - 1);
        } else {
          return mid;
        }
      }

Hash table

A hash table is a data structure that stores data in key-value pairs. The key is used to look up the value. The hash table is a good data structure to use when you need to look up values quickly.

Sorting

Sorting is the process of rearranging the elements of an array so that they are in some kind of order. The most common sorting algorithms are bubble sort, selection sort, insertion sort, and merge sort.

The following code demonstrates the bubble sort algorithm:

    
      function bubbleSort(arr) {
        var swapped;
        do {
          swapped = false;
          for (var i = 0; i < arr.length; i++) {
            if (arr[i] > arr[i + 1]) {
              var temp = arr[i];
              arr[i] = arr[i + 1];
              arr[i + 1] = temp;
              swapped = true;
            }
          }
        } while (swapped);
        return arr;
      }
