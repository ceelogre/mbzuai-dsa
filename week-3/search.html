Searching & Sorting Algorithms

There are mainly two types of search:

  • Linear search
  • Binary search

Linear search is a simple search algorithm that checks each element of a list sequentially until it finds the target value.

Binary search is a search algorithm that works by comparing the target value to the middle element of the list. If the target value is equal to the middle element, then the position is found. Otherwise, if the target value is lower than the middle element, then the search continues on the lower half of the list. If the target value is greater than the middle element, then the search continues on the upper half of the list. It is used when the list is sorted.

The following code demonstrates the linear search algorithm:

    
      function linearSearch(arr, target) {
        for (var i = 0; i < arr.length; i++) {
          if (arr[i] === target) {
            return i;
          }
        }
        return -1;
      }

The following code demonstrates the binary search algorithm:

    
      function binarySearch(arr, target) {
        var low = 0;
        var high = arr.length - 1;
        var mid;
        while (low <= high) {
          mid = Math.floor((low + high) / 2);
          if (arr[mid] < target) {
            low = mid + 1;
          } else if (arr[mid] > target) {
            high = mid - 1;
          } else {
            return mid;
          }
        }
        return -1;
      }

The following code demonstrates the binary search algorithm with a recursive function:

    
      function binarySearch(arr, target, low, high) {
        if (low > high) {
          return -1;
        }
        var mid = Math.floor((low + high) / 2);
        if (arr[mid] < target) {
          return binarySearch(arr, target, mid + 1, high);
        } else if (arr[mid] > target) {
          return binarySearch(arr, target, low, mid - 1);
        } else {
          return mid;
        }
      }

Hash table

A hash table is a data structure that stores data in key-value pairs. The key is used to look up the value. The hash table is a good data structure to use when you need to look up values quickly.

Sorting

Sorting is the process of rearranging the elements of an array so that they are in some kind of order. The most common sorting algorithms are bubble sort, selection sort, insertion sort, and merge sort.

Types of sorting algorithms

Bubble sort is a sorting algorithms works by repeatedly swapping adjacent values until the whole list is sorted.

The following code demonstrates the bubble sort algorithm:

    
      function bubbleSort(arr) {
        var swapped;
        do {
          swapped = false;
          for (var i = 0; i < arr.length; i++) {
            if (arr[i] > arr[i + 1]) {
              var temp = arr[i];
              arr[i] = arr[i + 1];
              arr[i + 1] = temp;
              swapped = true;
            }
          }
        } while (swapped);
        return arr;
      }

Selection sort works by selecting the smallest item in each iteration and placing it at the beginning of an unsorted list.

The following code demonstrates the selection sort algorithm:

    
      function selectionSort(arr) {
        var min;
        for (var i = 0; i < arr.length; i++) {
          min = i;
          for (var j = i + 1; j < arr.length; j++) {
            if (arr[j] > arr[min]) {
              min = j;
            }
          }
          if (min !== i) {
            var temp = arr[i];
            arr[i] = arr[min];
            arr[min] = temp;
          }
        }
        return arr;
      }

Quick sort works by selecting a pivot element then sorting items on its left and right. The process is repeated until the whole list is sorted.

The following code demonstrates quick sort algorithm

    
      function quickSort(arr) {
        if (arr.length <= 1) {
          return arr;
        }
        var pivot = arr.shift();
        var left = [];
        var right = [];
        for (var i = 0; i < arr.length; i++) {
          if (arr[i] < pivot) {
            left.push(arr[i]);
          } else {
            right.push(arr[i]);
          }
        }
        return quickSort(left).concat(pivot, quickSort(right));
      }

The following code demonstrates the insertion sort algorithm:

    
      function insertionSort(arr) {
        for (var i = 1; i < arr.length; i++) {
          var temp = arr[i];
          for (var j = i - 1; j >= 0 && arr[j] > temp; j--) {
            arr[j + 1] = arr[j];
          }
          arr[j + 1] = temp;
        }
        return arr;
      }

The following code demonstrates the merge sort algorithm:

    
      function mergeSort(arr) {
        if (arr.length === 1) {
          return arr;
        }
        var mid = Math.floor(arr.length / 2);
        var left = arr.slice(0, mid);
        var right = arr.slice(mid);
        return merge(mergeSort(left), mergeSort(right));
      }
      function merge(left, right) {
        var result = [];
        while (left.length && right.length) {
          if (left[0] <= right[0]) {
            result.push(left.shift());
          } else {
            result.push(right.shift());
          }
        }
        while (left.length) {
          result.push(left.shift());
        }
        while (right.length) {
          result.push(right.shift());
        }
        return result;
      }
