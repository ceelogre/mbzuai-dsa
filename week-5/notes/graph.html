Graph Data Structure

A graph is a type of data structure in which a group of nodes are interconnected via edges.

Graph Terminologies

A directed graph is a graph in which edges are labeled with a direction. A directed edge from node A to node B is denoted as A->B.

An undirected graph is a graph in which edges are not labeled with a direction. An undirected edge from node A to node B is denoted as A-B.

Adjacency: The relationship between two nodes is called adjacency. Two nodes are adjacent if they are connected by an edge.

Path: A path is a sequence of nodes that are connected by edges.

Cycle: A cycle is a path that starts and ends at the same node.

Degree: The number of edges connected to a node.

Graph representation

A graph is represented by a set of nodes and a set of edges. Each node is represented by a unique integer.

Each edge is represented by a pair of nodes. The nodes are connected by an edge.

Graphs can be represented in mostly two ways:

  • Adjacency List
  • Adjacency Matrix

Adjacency List

Adjacency list is a list of nodes and a list of edges. Each node is represented by a unique integer.

Adjacency Matrix

Adjacency matrix is a matrix of nodes and a matrix of edges. Each node is represented by a unique integer.
