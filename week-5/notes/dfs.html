Depth First Search Algorithm

Often abbreviated as DFS, the depth-first search algorithm is a graph traversal algorithm that explores all paths between any two given vertices. The algorithm is also known as a graph-search algorithm because it is used to search for a path in a graph.

The algorithm is recursive and has the following steps:

  1. Mark the current node as visited.
  2. Explore all the neighbors of the current node.
  3. If the current node has no unvisited neighbors, backtrack to the previous node.

Example

Here's an implementation of the DFS algorithm in JavaScript:


    function dfs(graph, start) {
      const visited = {};
      const stack = [start];
      while (stack.length) {
        const node = stack.pop();
        if (!visited[node]) {
          visited[node] = true;
          stack.push(...graph[node]);
        }
      }
      return visited;
    }

After finding all connected graph nodes, it becomes easy to perform other operations like find a path between two nodes and finding the shortest path among others.
