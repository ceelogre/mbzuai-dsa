Queue Data Structure

A queue is a linear data structure where insertions occur at the front and removal at the end

A queue is a FIFO data structure, meaning that the first element inserted is the first element removed

There are different types of queue data structure.

  • Simple queue
  • Circular queue
  • Priority queue
  • Deque

Basic operations of queue

  • Enqueue
  • Dequeue
  • Peek
  • IsEmpty
  • IsFull
