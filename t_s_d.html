Tuples, Sets and Dictionaries

Tuples are types of sequence types that consist of a number of values separated by commas.

address = 'kk', 8, 'ave'

Tuples are immutable

To create an empty or a 1 item tuple use:

t = () t = (1,)

To create a tuple with a single value use:

t = (1, 2, 3)

To unpack values from a tuple, use:

a,b,c = t

Sets are types of unordered collections of unique values.

heights = {15, 27, 7, 28,15, 3}

Sets support mathematical operations like and, or, intersection and so fort.

Checkout examples in the IDE.

Dictionaries are types of mappings from keys to values.

id = {'Diane': 21, 'university': 'MBZUAI', 'weight': 128, 101: 'Data structures'}

To access a value from a dictionary, use:

id['Diane']

To add a new key-value pair to a dictionary, use:

id['qualification'] = 'Masters'

To remove a key-value pair from a dictionary, use:

del id['Diane']

To check if a key is in a dictionary, use:

'Diane' in id

To check if a value is in a dictionary, use:

21 in id

To get the keys of a dictionary, use:

list(id)

To loop through items, use items()

Tuples are immutable, sets are mutable, and dictionaries are mutable.
