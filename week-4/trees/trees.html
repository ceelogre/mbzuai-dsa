Trees Data Structure

Trees are a type of non-linear data structure where data is arranged in a tree-like structure. They are used to store data in a hierarchical manner

Linear data structure like arrays pose a challenge because time complexity increases as the size increases.

  • The root of the tree is called the root node
  • The children of a node are called the child nodes
  • The next sibling of a node is called the next node
  • The parent of a node is called the parent node
  • The previous sibling of a node is called the previous node
  • The link between any two nodes is an edge
  • The leaves of a tree are called leaf nodes
  • A tree can be either unbalanced or balanced
  • A tree can be either complete or non-complete
  • A tree can be either full or non-full
  • A tree can be either perfect or non-perfect
  • A tree can be either symmetric or non-symmetric
  • A tree can be either dense or non-dense
  • A tree can be either sparse or non-sparse
  • A tree can be either doubly-linked or singly-linked
  • A tree can be either circular or non-circular

Tree traversal

There are three types of tree traversal:

  • Inorder traversal
  • Preorder traversal
  • Postorder traversal

Tree traversal - Inorder traversal

Inorder traversal is done by recursively traversing the left subtree, printing the root and then traversing the right subtree.

    
      // Inorder traversal
      void inorder(Node *root) {
        if (root != NULL) {
          inorder(root->left);
          cout << root->data << " ";
          inorder(root->right);
        }
      }
    
    5 12 6 1 9

Tree traversal - Preorder traversal

Pre-order traversal is done by visiting the root node, then the left subtree and finally the right subtree.

    
      // Preorder traversal
      void preorder(Node *root) {
        if (root != NULL) {
          cout << root->data << " ";
          preorder(root->left);
          preorder(root->right);
        }
      }
    
    1 12 5 6 9

Tree traversal - Postorder traversal

Post-order traversal is done by recursively traversing the left subtree, then the right subtree and finally printing the root node.

    
      // Postorder traversal
      void postorder(Node * root){
        if (root != Null) {
          postorder(root->left)
          postorder(root -> right)
          cout << root-> data << ' ';
        }
      }
    
    5 6 12 9 1
